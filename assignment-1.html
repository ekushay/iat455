Assignment 1

This assignment is broken into 3 seperate parts. Each part should be completed as an "Effect" that is loadable and runnable using the example code presented in your workshops.

Part 1: HSV -> RGB

Create a scanline effect like the one shown in labs, but allow the user to control the color of the scanline using HSV controls. You will have to take the HSV values, and translate them to RGB values before you can draw the scanlines.

Use the following scales for your HSV ranges:

  • Hue: 0 - 360
  • Saturation: 0 - 255
  • Value: 0 - 100

Part 2: Run Length Compression

Create an effect using the run length compression algorithm. Compress the image, then recreate the image data using the compressed information.

In order for the algorithm to actually compress anything use a threshold for pixel differences and round the values to their closest match.

EX: rgba(5, 18, 61, 128) -> rgba(0, 16, 64, 128)

This way similar pixels will be grouped together and we'll see some benefit to the compression at the expense of bit depth.

You can do this on a single channel if you find it easier, but show the resulting image as black and white, and add a control that allows the user to switch which channel is being used.

Part 3: Gaussian Blur

Create an effect that will apply a gaussian blur to an image and allow the user to control the radius of the blur.
